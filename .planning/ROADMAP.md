# Roadmap: AI Event Signup Page

## Overview

Build a high-converting signup page for the AI training event. Start with page structure and content, add the form with validation, then connect webhook and polish the experience.

- [ ] **Phase 1: Foundation** - Page route, hero, session info bar, benefits section

- [ ] **Phase 3: Integration** - Webhook submission, success/error states, polish
